# Правила игры GI Elements

## Концепция

GI Elements — это головоломка-пасьянс на шестиугольном поле с элементами Genshin Impact. Нужно убрать все элементы с поля, удаляя их парами.

## Элементы

Игра использует 7 элементов из мира Genshin Impact:

| Элемент | Описание |
|---------|----------|
| **Pyro** | Огонь |
| **Hydro** | Вода |
| **Electro** | Электро |
| **Cryo** | Крио |
| **Anemo** | Анемо |
| **Dendro** | Дендро |
| **Geo** | Гео |

### Бездна (опционально)

Универсальный элемент, который можно соединить с **любым** другим элементом. Может быть включена или отключена в настройках.

## Игровое поле

- Поле представляет собой шестиугольную сетку
- Игра начинается с поля размером **3** (19 клеток)
- Максимальный размер поля настраивается (от 3 до 9)
- При необходимости поле может расширяться или уменьшаться

## Ходы

### Удаление пар

Два элемента можно удалить с поля, если:

1. Они **одинаковые** ИЛИ один из них — **Бездна**
2. Они расположены на **одной прямой** (горизонталь, вертикаль или диагональ шестиугольника)
3. Между ними **нет других элементов** (все промежуточные клетки пусты)

### Порядок действий

1. Нажмите на первый элемент — он подсветится
2. Нажмите на второй элемент
3. Если пара валидна — оба элемента удаляются с поля
4. Если пара не валидна — выбор сбрасывается

## Кнопки управления

| Кнопка | Иконка | Действие |
|--------|--------|----------|
| Новая игра | 🔄 | Начать игру заново |
| Настройки | ⚙️ | Открыть панель настроек |
| Правила | ❓ | Показать правила игры |
| Подсказка | 💡 | Подсветить доступную пару элементов |
| Отмена | ↩️ | Отменить последний ход |
| Добавить | ➕ | Добавить новые элементы на поле |

## Добавление элементов

Когда на поле не осталось доступных пар для удаления, кнопка **«Добавить»** становится активной.

При нажатии:
1. Если есть **скрытые элементы** — они добавляются первыми
2. Иначе элементы, имеющиеся на доске, собираются и перемешиваются
3. Если внешний уровень поля пуст и во внутреннем поле достаточно места — поле **уменьшается**
4. Если пустых клеток недостаточно — поле **расширяется** на один уровень
5. Пустые клетки заполняются элементами (начиная с внешних уровней)

### Переполнение поля

Если максимальный размер поля достигнут, но элементов для добавления больше, чем пустых клеток:
- Заполняются только доступные пустые клетки
- **Оставшиеся элементы сохраняются** и будут использованы при следующем дополнении
- Количество скрытых элементов отображается в поле **«Осталось»** в скобках: например, `15 (3)`

## Настройки

| Параметр | Описание |
|----------|----------|
| **Макс. сторона поля** | Максимальный размер стороны поля (3–9) |
| **Бездна** | Включение/выключение универсального 8-го элемента |
| **Подсказка** | Включение/выключение кнопки помощи |

Настройки сохраняются автоматически и восстанавливаются при следующем запуске.

## Победа

Игра считается выигранной, когда на поле остаётся **один** элемент.
