# GI Elements

Шестиугольный пасьянс с элементами Genshin Impact. Головоломка, в которой нужно убрать все элементы с поля, удаляя их парами.

## Правила игры

[Полные правила игры →](public/rules.md)

## Особенности

- **Шестиугольное поле** — элементы расположены на гексагональной сетке
- **8 элементов** — Pyro, Hydro, Electro, Cryo, Anemo, Dendro, Geo + Бездна (опционально)
- **Динамическое поле** — начинается со стороны 3, расширяется и уменьшается автоматически
- **Очередь элементов** — при переполнении поля скрытые элементы сохраняются для следующего дополнения
- **Настройки** — максимальный размер поля, включение Бездны, кнопка подсказки
- **Отмена ходов** — возможность вернуть последний ход

## Технологии

- Vue 3 (Composition API)
- Vite
- SVG-иконки элементов
- FontAwesome (иконки кнопок)
- CSS-анимации и градиенты в стиле Genshin Impact

## Запуск

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для продакшена
npm run build
```

## Структура проекта

```
src/
├── assets/elements/       # SVG-иконки элементов
├── components/
│   ├── HexGrid.vue        # Шестиугольная сетка
│   ├── GameOverlay.vue    # Экран победы
│   ├── SettingsModal.vue  # Модалка настроек
│   └── RulesModal.vue     # Модалка правил
├── composables/
│   ├── useGameState.js    # Состояние игры
│   ├── useGameLogic.js    # Логика проверки пар
│   └── useSettings.js     # Управление настройками
├── pages/
│   └── IndexPage.vue      # Главная страница
└── utils/
    ├── hex.js             # Утилиты для гексов
    ├── elements.js        # Данные элементов
    └── constants.js       # Константы
```
